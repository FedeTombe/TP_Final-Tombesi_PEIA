# TP3 - Probabilidad y Estadística para la Inteligencia Artificial

Autor: Federico Tombesi

## Descripción

Este repositorio contiene la resolución del Trabajo Práctico 3 correspondiente a la materia Probabilidad y Estadística para la Inteligencia Artificial.

El proyecto incluye:

- Simulación de clientes mediante distribuciones de Poisson.
- Estimación de intervalos empíricos.
- Análisis de la varianza (ANOVA).
- Prueba t de Student.
- Generación de gráficos y estadísticas descriptivas.

## Requisitos

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python main.py
```
